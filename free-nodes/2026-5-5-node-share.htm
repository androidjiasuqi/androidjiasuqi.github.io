第一章：认识天眼科学上网

1.1 什么是科学上网？

科学上网本质上是一种通过加密隧道技术绕过网络审查的方法。它像一条看不见的数字地道，将您的网络请求从受限制的本地网络引导至境外的自由节点，再访问目标网站，最后将获取的内容原路返回给您。这种技术巧妙地避开了本地网络的监控和过滤系统。

1.2 天眼科学上网的独特优势

与传统VPN相比，天眼科学上网在三个方面实现了质的飞跃：

速度革命：采用智能路由算法，自动选择最优服务器节点，将延迟降低40%以上。测试数据显示，在高峰时段观看1080p视频的缓冲时间不超过2秒。

隐匿性突破：运用最新的Obfsproxy混淆技术，使VPN流量与普通HTTPS流量几乎无法区分，有效规避深度包检测(DPI)系统的识别。

全平台覆盖：从Windows到MacOS，从Android到iOS，甚至支持路由器级别的全局部署，真正实现"一次配置，全家畅享"。

3.2 协议选择指南

不同场景下应选用不同协议：
- WireGuard：速度最快，适合4K视频串流
- OpenVPN TCP：稳定性最佳，重要会议时首选
- Shadowsocks：混淆能力最强，在严格审查地区效果显著

3.3 高级功能详解

分流模式：
通过自定义规则，让国内流量直连，国外流量走代理，既节省服务器资源又提升访问速度。建议将微信、支付宝等加入直连名单。

Kill Switch：
网络中断时立即切断所有连接，防止真实IP泄露。建议在敏感操作时强制启用。

FAQ

V2RayN 的作用是什么？
V2RayN 是 V2Ray 的 Windows 图形客户端，提供节点管理、订阅导入、日志查看等功能。它支持多协议配置，并可与系统代理集成，是 Windows 用户最常用的代理管理工具之一。
Matsuri 节点延迟排序功能如何使用？
Matsuri 会定期测速节点延迟，并结合策略组自动选择延迟最低节点。用户无需手动切换即可保持高速访问，优化网页、视频和游戏体验，提高网络稳定性。
Matsuri 如何提升节点连接效率？
Matsuri 是一个支持多协议的代理工具，强调快速连接与自动切换功能。它可以根据延迟自动选择最优节点，减少手动切换的繁琐过程，保证网络体验的流畅性。
Hiddify Next 的多机场订阅如何整合？
Hiddify Next 提供合并订阅功能，可将多个链接导入同一配置，自动去重并分类节点。用户可设置优先级与标签，方便快速选择不同提供商的线路，提升连接灵活性与冗余度。
如何在 V2Ray 中自定义路由规则？
在 V2Ray 中可以通过修改 config.json 文件中的 routing 部分来自定义路由规则。用户可以根据 IP 段、域名或地理位置来决定流量的去向，从而实现自动分流或访问加速的效果。
Mellow 的 TUN 模式与 SOCKS5 代理区别是什么？
TUN 模式拦截系统 IP 层流量，实现全局代理；SOCKS5 代理通常只影响特定应用。TUN 更适合全局加速、游戏和视频应用，保证所有流量走节点，提高网络稳定性和速度。
WinXray 的节点导入方式有哪些？
WinXray 支持手动添加节点、从 URL 订阅导入节点，以及批量导入 JSON 配置文件。用户可以直接复制订阅地址或粘贴节点链接，程序会自动解析并添加到节点列表中，操作十分便捷。
Hiddify Next 订阅更新后节点消失，如何恢复？
先检查订阅源是否更改格式或被下架，尝试重新粘贴订阅链接并手动刷新；若仍缺失，联系订阅提供方确认是否停用某些节点或使用了新的访问权限机制（例如 token 验证）。
Shadowrocket 在使用订阅时要注意什么？
在 Shadowrocket 中添加订阅时，应确保链接以 https 开头且格式正确。此外，若节点更新失败，可尝试关闭缓存或重新获取订阅链接，以保证配置文件同步最新节点数据。
如何将 Clash 节点导入到 Sing-box？
Sing-box 支持从 Clash 配置文件中导入节点。用户只需将 YAML 文件放入配置目录或通过命令导入即可。系统会自动解析规则组和节点信息，简化迁移过程。
